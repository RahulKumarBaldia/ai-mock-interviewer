# 🎤 AI Mock Interviewer

🚀 **Live Demo:** Click Here

An AI-powered mock interview platform designed to help aspiring web developers prepare for technical interviews. Users can customize their interview by selecting a web development role, target company, and difficulty level. After completing the interview, the AI provides detailed feedback, a performance score, strengths, areas for improvement, an honest hiring verdict, and question-by-question analysis to help users improve their interview skills.

## ✨ Features

- 🤖 AI-generated mock interviews
- 💼 Choose your target Web Development role
- 🏢 Select your preferred company
- 📊 Choose interview difficulty level
- 📝 AI-powered interview questions
- 💬 Detailed AI feedback
- 📈 Overall interview score
- 💪 Strengths analysis
- 📌 Areas for improvement
- ⚖ Honest hiring verdict
- 📋 Question-by-question review

## 🚀 Getting Started

### Prerequisites

- Node.js Installed
- MongoDB Installed
- Groq API Key

### Installation

Clone the repository

```bash
git clone https://github.com/RahulKumarBaldia/ai-mock-interviewer.git

cd ai-mock-interviewer
```

### Setup Backend

```bash
cd Backend

npm install
```

Create a `.env` file inside the **Backend** folder

```env
PORT=5000
MONGO_URI=your_mongodb_connection_string
GROQ_API_KEY=your_groq_api_key
```

Start the backend server

```bash
node server.js
```

### Setup Frontend

```bash
cd ../Frontend

npm install
```

Start the frontend

```bash
npm run dev
```
